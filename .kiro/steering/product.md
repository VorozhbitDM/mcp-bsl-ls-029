# Продукт: MCP BSL Language Server

MCP-сервер (Model Context Protocol), который связывает AI-ассистентов с **BSL Language Server** — инструментом статического анализа и форматирования кода 1С:Предприятие (BSL).

## Назначение

Предоставляет два MCP-инструмента для AI-агентов:
- **`bsl_analyze`** — запускает статический анализ `.bsl`/`.os` файлов, возвращает ошибки, предупреждения и подсказки от BSL Language Server
- **`bsl_format`** — форматирует BSL-файлы согласно настроенным правилам стиля

## Основной сценарий использования

AI-агенты, работающие с проектами на 1С:Предприятие, используют этот сервер для проверки и форматирования BSL-кода без выхода из рабочего процесса. Сервер оборачивает JAR-файл BSL Language Server и преобразует его JSON-вывод в структурированные MCP-ответы.

## Целевая аудитория

Разработчики конфигураций 1С:Предприятие, использующие AI-ассистентов (Cursor, Kiro и др.) и желающие интегрировать автоматическую проверку синтаксиса и форматирование кода в свой AI-рабочий процесс.

## Ключевые ограничения

- Требуется JDK 17+ (для запуска JAR-файла BSL Language Server)
- Требуется Python 3.10+
- Ориентирован на Windows (пути используют Windows-соглашения; безопасное окружение перенаправляет APPDATA/TEMP для обхода проблем с junction point)
- JAR-файл BSL Language Server должен присутствовать локально; путь задаётся через переменную окружения `BSL_JAR`
